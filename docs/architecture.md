# CPU Image Gen 统一架构文档

## 架构概览

```
┌─────────────────────────────────────────────────────────────────┐
│                        用户层 (User Layer)                       │
│                                                                 │
│  Layer 1: 基础    Layer 2: 控制    Layer 3: 高级    Layer 4: 专家│
│  ─────────────   ─────────────   ─────────────   ─────────────  │
│  输入描述         +风格/模板       +参数调整        +完全掌控     │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                        功能层 (Function Layer)                   │
│                                                                 │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐          │
│  │ 输入解析  │ │ 文化实体  │ │ 后处理    │ │ 批量处理  │          │
│  │          │ │ 场景模板  │ │ 图片放大  │ │ 模型缓存  │          │
│  │          │ │ 风格迁移  │ │ 负面提示  │ │          │          │
│  └──────────┘ └──────────┘ └──────────┘ └──────────┘          │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                        技术层 (Technology Layer)                 │
│                                                                 │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐          │
│  │ 中文NLP   │ │ 知识图谱  │ │ 图像处理  │ │ 并行计算  │          │
│  │          │ │ 文化数据库│ │ OpenCV   │ │ 批量优化  │          │
│  │          │ │          │ │ PIL      │ │          │          │
│  └──────────┘ └──────────┘ └──────────┘ └──────────┘          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                        模型层 (Model Layer)                      │
│                                                                 │
│  CPU 生图        GPU 生图         云 GPU          未来模型      │
│  ─────────────  ─────────────   ─────────────  ─────────────  │
│  SDXL Turbo     SDXL/SD3        云端推理        Flux/中文模型   │
└─────────────────────────────────────────────────────────────────┘
```

## 功能模块清单

| 模块 | 功能 | 技术 | 依赖 | 状态 |
|------|------|------|------|------|
| 输入解析 | 中文→结构化 | NLP、正则 | - | ✓ |
| 文化实体 | 70个文化元素 | 知识图谱 | 输入解析 | ✓ |
| 场景模板 | 23个预设场景 | 模板引擎 | 输入解析 | ✓ |
| Prompt构建 | 组合生成 | 规则引擎 | 文化实体、场景模板 | ✓ |
| 图像生成 | 图片生成 | SDXL/Flux | GPU/CPU | ✓ |
| 图片放大 | 分辨率提升 | OpenCV | 图像生成 | ✓ |
| 负面提示 | 排除控制 | Prompt编辑 | Prompt构建 | ✓ |
| 批量处理 | 批量生成 | 并行计算 | 图像生成 | ✓ |
| API服务 | 接口暴露 | Flask | 所有模块 | ✓ |
| Web UI | 用户界面 | React | API服务 | ✓ |

---

## 分层控制

| 层级 | 名称 | 控制点 | 学习价值 |
|------|------|--------|----------|
| Layer 1 | 基础 | 输入、生成 | prompt 结构 |
| Layer 2 | 控制 | +风格、模板 | 风格影响 |
| Layer 3 | 高级 | +步数、分辨率、负面提示 | 参数控制 |
| Layer 4 | 专家 | +模型、放大、全部参数 | 完全掌控 |
