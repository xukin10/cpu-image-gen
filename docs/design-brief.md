# CPU Image Gen - Web 设计需求文档

## 项目概述

**项目名称：** CPU Image Gen - AI 图片生成工具

**项目定位：** 一个支持中文自然语言输入的 AI 图片生成工具，让任何人都能轻松生成高质量图片。

**目标用户：**
- 普通用户：想试试 AI 生图
- 设计师：快速生成素材
- 内容创作者：批量生成图片

---

## 设计风格

### 核心风格：Linear 风格（简约优雅）

**设计原则：**

| 原则 | 说明 | 具体做法 |
|------|------|----------|
| 质感 | 看起来专业、精致 | 深色背景 + 微妙渐变 + 精致边框 |
| 动效 | 有反馈但不过度 | 悬停反馈 + 加载动画 + 页面过渡 |
| 排版 | 层次分明、易读 | 大标题 + 适中正文 + 大量留白 |
| 克制 | 每个元素都有意义 | 不过度装饰，功能优先 |

**参考网站：** https://linear.app

**功能：** 用户输入描述，生成图片

**布局：**
```
┌─────────────────────────────────────────────────────────┐
│ Logo                                    [历史] [设置]    │
├─────────────────────────────────────────────────────────┤
│                                                         │
│                   用中文描述你想要的图片                  │
│                   ─────────────────────                 │
│                                                         │
│  ┌─────────────────────────────────────────────────┐   │
│  │  例：赛博朋克城市夜景，雨天霓虹灯                 │   │
│  └─────────────────────────────────────────────────┘   │
│                                                         │
│                       [ 生成图片 ]                       │
│                                                         │
│  ──────────────────────────────────────────────────    │
│                                                         │
│  快速开始                                               │
│  [风景] [人像] [动漫] [产品] [更多]                     │
│                                                         │
├─────────────────────────────────────────────────────────┤
│  最近生成                                               │
│  [图1] [图2] [图3] [图4]                                │

**交互要点：**
- 输入框：聚焦时边框发光，有占位提示
- 生成按钮：悬停微动，点击变加载状态
- 快捷模板：点击直接填入示例

## 配色方案

```

卡片背景：#14141f (深灰)
边框：#1e1e2e (微妙灰)

## 组件规范

### Button

| 状态 | 样式 |
|------|------|
| 默认 | 背景 #6366f1，文字白色，圆角 8px |
| 悬停 | 背景 #818cf8，微上移 2px |

| 状态 | 样式 |
|------|------|

| 状态 | 样式 |
|------|------|
| 默认 | 背景 #14141f，边框 #1e1e2e |

## 交互规范

### 页面过渡
- 使用淡入淡出，时长 200ms
- 使用 ease-out 缓动

### 悬停反馈
- 按钮：颜色变化 + 微位移
- 卡片：边框变化 + 阴影
- 链接：颜色变化

### 加载状态
- 骨架屏：闪烁动画
- 进度条：线性进度
- 按钮：旋转图标
